Frequently Asked Questions

Everything about using Terse with browser-based AI.

How does Terse work with ChatGPT?
Terse uses macOS Accessibility APIs to read text from your browser's ChatGPT textarea. When you focus the tab, a floating bar shows the compressed version. One click replaces your draft, or Auto mode compresses every message automatically.
Does Terse need a browser extension?
No. Terse is a native macOS app using Accessibility APIs — the same technology as screen readers. No browser extension required. Works with Chrome, Safari, and all macOS browsers.
Does it work with Claude.ai and Gemini too?
Yes. Terse works with any browser-based AI assistant using a standard textarea, including ChatGPT, Claude.ai, Gemini, and Perplexity. The same floating bar appears automatically for each.

Is Terse safe to use with ChatGPT?
Yes. All text processing is 100% on-device using a Rust engine. Nothing is sent to Terse's servers. Your prompts and conversations remain completely private.
