Team tokens —

Paste a token into each developer's Terse client (Mac, Windows, Chrome, VS Code) to start collecting analytics.

Client setup

Each developer pastes a team token into their Terse client. The token is shown once on creation — copy from the row above or mint a fresh one.

# Mac / Windows (Tauri)
mkdir -p ~/.terse && echo '{"team_token":"YOUR_TOKEN","user_email":"you@company.com"}' > ~/.terse/cloud.json

# VS Code — Preferences → Settings → search "Terse"
# Terse: Team Token   →  paste token
# Terse: Team Email   →  your email

# Chrome extension — Options → Terse Cloud → paste token + email
