产品 02
四层产品，
一次安装。
Terse 以轻量级系统层运行，无需云端代理，无需 API Key。支持 macOS、Windows 桌面端，以及 VS Code / Cursor 浏览器插件。在发送给 AI 之前，静默压缩、监控、管理每一条消息。
层 01 — 优化器（跨平台）
Token 压缩引擎
macOS 悬浮窗 + Windows 系统托盘 + VS Code / Cursor 插件，三端统一。发送前静默压缩：轻度（拼写纠错+空白压缩）、标准（去填充词、缩短短语）、激进（缩写+去 Markdown）。20+ 压缩技术，完全本地运行，零延迟。

竞争优势与团队 05
难以复制，
容易留存。
本地压缩引擎 + 实时 Agent 日志解析 + 双边 Token 市场 + 企业合规部署，四层能力的组合，任何单一功能产品都无法在 12 个月内追平。
  • 本地优先架构。 数据不出本地，零云端延迟。对合规要求严格的企业（金融、法律、政府）是硬性准入条件，也是销售中最强的差异化论据。
  • 深度 Agent 集成。 读取 Claude Code JSONL 日志、Cursor SQLite 数据库、Codex Hook 输出，每一个 Agent 的接入都需要逆向工程，竞争对手需要重复相同的工作量。
  • 交易所网络效应。 买方带来卖方，流动性自我增强。$50 万 GMV 以上的双边市场极难被替代。
  • 11 语言国际化。 已上线中、英、日、韩、阿、法、德、西、葡、意，西方同类工具几乎不覆盖亚太市场，Terse 在中国及东南亚几乎没有直接竞争者。
  • 插件生态入口。 VS Code 插件市场拥有 3,500 万活跃用户。插件免费版即是病毒式分发渠道，也是企业客户导入的低摩擦入口。
  • 自研智能路由引擎（数据飞轮壁垒）。 基于自主微调的 DistilBERT 模型，实时对每条 Prompt 进行复杂度分类（验证集准确率 98.16%），将简单请求自动路由至低成本模型（如 claude-opus-4-6 → claude-sonnet-4-6，gpt-4o → gpt-4o-mini），平均节省 60–80% 推理成本，用户无感知。训练数据融合 Code Alpaca、Alpaca 52k、CodeSearchNet 及手工标注锚样本；随着用户规模扩大，真实 Prompt 持续回流训练集，模型识别精度将螺旋式提升——竞争对手越晚入场，数据差距越难追平。量化部署（INT8 ONNX，64 MB），本地推理延迟 <80 ms，全程离线，无隐私泄露风险。
