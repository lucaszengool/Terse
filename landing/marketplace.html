Quick Start

Use your Terse key exactly like the original API. Just change the base URL:

Python (Anthropic)
# pip install anthropic import anthropic client = anthropic.Anthropic( api_key="terse_bk_your_key_here", base_url="https://www.terseai.org/api/proxy", ) msg = client.messages.create( model="claude-sonnet-4-20250514", max_tokens=1024, messages=[{"role": "user", "content": "Hello!"}] )
Python (OpenAI)
# pip install openai from openai import OpenAI client = OpenAI( api_key="terse_bk_your_key_here", base_url="https://www.terseai.org/api/proxy/v1", ) resp = client.chat.completions.create( model="gpt-4o", messages=[{"role": "user", "content": "Hello!"}] )
cURL
curl https://www.terseai.org/api/proxy/v1/messages \ -H "Authorization: Bearer terse_bk_your_key_here" \ -H "Content-Type: application/json" \ -H "anthropic-version: 2023-06-01" \ -d '{"model":"claude-sonnet-4-20250514","max_tokens":1024,"messages":[{"role":"user","content":"Hello!"}]}'
